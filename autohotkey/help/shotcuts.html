This help contains all shortcuts created/overrided by AutoHotkey app.

Version 2 script is uses

Global

Location of file: $HOME/.config/autohotkey/global.ahk

Command Execution Remarks
+ q Close current app
+ f Toggle current app fullscreen
+ o Open Select Area to Crop screenshot
+ h Open This help page
+ l Lock screen

Glazewm

Location of file: $HOME/.glaze-wm/config.yml

Command Execution Remarks
+ h / + left Focus application on left
+ l / + right Focus application on right
+ j / + down Focus application on down
+ k / + up Focus application on up
+ + h / + + left Move application to left
+ + l / + + right Move application to right
+ + j / + + down Move application to down
+ + k / + + up Move application to up
+ a Move workspace to left monitor
+ s Move workspace to down monitor
+ d Move workspace to up monitor
+ f Move workspace to right monitor
+ 1...0 Go to workspace 1...0
+ + 1...0 Move to workspace 1...0
+ u Decrease vertical size
+ i Decrease horizontal size
+ o Increase horizontal size
+ p Increase vertical size
+ r Go to resize mode Once in resize mode, use arrows or hjkl to increase/decrease size
(inside resize mode) esc / enter / + r Go back to normal mode
+ v Toggle tiling direction
+ + space Toggle floating
+ + e Close Glazewm
+ + r Reload Glazewm

Apps

Location of file: $HOME/.config/autohotkey/apps.ahk

Command Execution Remarks
+ Open Terminal WSL2 Linux using Windows Terminal App
+ + Open Powershell
+ + Open Command Prompt
+ b Open Browser Chrome
+ + b Open Browser Chrome (Incognito)
+ + y Open Invidio.us instance in Browser Chrome (Incognito), https://inv.tux.pizza/
+ + u Open QMusic in Browser Chrome (Incognito), https://qmusic.be/luister/qmusic_be
