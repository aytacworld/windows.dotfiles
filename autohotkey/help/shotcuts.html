This help contains all shortcuts created/overrided by AutoHotkey app.

Version 2 script is uses

Global

Location of file: $HOME/.config/autohotkey/global.ahk

Command Execution Remarks
+ q Close current app
+ f Toggle current app fullscreen
+ o Open Select Area to Crop screenshot
+ h Open This help page

Apps

Location of file: $HOME/.config/autohotkey/apps.ahk

Command Execution Remarks
+ Open Terminal WSL2 Linux using Windows Terminal App
+ + Open Powershell
+ + Open Command Prompt
+ b Open Browser Chrome
+ + b Open Browser Chrome (Incognito)
+ + y Open Invidio.us instance in Browser Chrome (Incognito), https://inv.tux.pizza/
+ + u Open QMusic in Browser Chrome (Incognito), https://qmusic.be/luister/qmusic_be

Komorebi (window tiling manager)

Location of file: $HOME/.config/autohotkey/komorebi.ahk

Command Execution Remarks
Generic
+ s apply custom-layout resizing doesn't work on custom layout
+ + o Reload komorebi
+ + r Retile Use this when tiling of the windows isn't working as expected
+ p Pause/Resume komorebi
+ t Toggle float
+ + f Toggle monocle Same as fullscreen, but via komorebi
Focus
+ h Focus on application left
+ j Focus on application down
+ k Focus on application up
+ l Focus on application right
+ 1 ... 0 Focus on workspace 1 ... 0 1 ... 0 means 1, 2, 3, ... 8, 9, 0. 0 being 10
Move
+ + h Move focused application to left
+ + j Move focused application to down
+ + k Move focused application to up
+ + l Move focused application to right
+ + 1 ... 0 Move focused application to workspace 1 ... 0 1 ... 0 means 1, 2, 3, ... 8, 9, 0. 0 being 10
+ x Flip windows horizontaly Doesn't work on custom layouts
+ y Flip windows verticaly Doesn't work on custom layouts
Resize
+ + h Resize horizontaly, decrease
+ + j Resize verticaly, increase
+ + k Resize verticaly, decrease
+ + l Resize horizontal, increase
